<h1> Challenges</h1>
One of the challenges I faced was creating two pages, one for my Part A and one for my Part B. I wanted a button at the bottom of the page that connected the two pages. But I had no idea what to do to make this. But then I remembered what we did for the BR restaurant guide and how we linked different files together. This is what the code that inspired me looked like:

```html
[greek](greek.md)
```

 This set me on the right track for creating a new page. So this would link two things together, and it wouldn't be hard to make a button, but I still didn't have a new page. So I made a new file, and I linked Part A to Part B. Then I looked up how to make a button, and this is what the code looks like:
```html
<a href="part-b.html" class="btn btn-primary">View Part B</a>
```
Another challenge I faced relating to this is that after I made this new file, I forgot to add all the starter code from the part A file. This made it so when I was trying to copy and paste my navbarcode it wouldn't work. I had both Part A and Part B side by side on my computer, trying to spot the difference between the two, when I realized that I needed the starter code. This allowed my navbar to work and all the other Bootstrap components that I was trying to use. 

Another challenge I faced was trying to link my A-frame to my part B. I wasn't sure if I would have to do something special here or if I could just link it as I linked Part A to Part B. It was actually fairly simple. I just created a new file, put my A-frame code into it, and made a button that linked to that file in my part B. This is what the code looks like:
```html
<div class="box bottom-box border border-dark flex-grow-1" style="min-height: 250px;">
<a href="aframe.html" class="btn btn-primary">veiw A-Frme</a>
```
I used the same code that I used to make the other button to do this one. And then I messed around with the sizing a little bit until I got it to something that I thought looked good.
